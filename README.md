## Getting Started
1. Use NodeJS LTS (current: v.20.x.x).
    - Verify using `nvm current` command.
2. Run development server:
    ```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    ```
3. Open [http://localhost:3000](http://localhost:3000)
